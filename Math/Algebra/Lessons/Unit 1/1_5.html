Lesson 5: Solving Equations with Variables on Both Sides

If $7x = 4x + 15$, what is the value of $x$?

If $9 - 2x = 3x - 1$, what is the value of $x$?

If $5x + 3 = 2x + 18$, what is the value of $3x$?

If $12 - 4m = m - 8$, what is the value of $m + 2$?

If $\frac{2}{3}k + 5 = \frac{1}{3}k + 9$, what is the value of $k$?

If 4(x−2)=2x+10, what is the value of x?

Two competing phone plans model their monthly charges using linear expressions. Plan A charges $C = 0.15m + 20$ and Plan B charges $C = 0.10m + 25$, where $m$ represents the minutes used. For how many minutes will the monthly charges of both plans be exactly equal?

If $\frac{4x - 3}{2} = 3x + 4$, what is the value of $2x$?

If $ax + b = cx + d$, where $a$, $b$, $c$, and $d$ are constants and $a \neq c$, which of the following expressions represents the value of $x$?

If $3(x - a) = 5(x + b)$, where $a$ and $b$ are constants, what is the value of $2x$ in terms of $a$ and $b$?
