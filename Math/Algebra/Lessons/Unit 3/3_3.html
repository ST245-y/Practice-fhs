Lesson 3: Reading and Interpreting Linear Function Graphs

A line is graphed in the standard $xy$-plane. The line passes through the origin $(0,0)$ and moves downward from left to right. Which of the following must be true about the equation of this line?

The graph below shows the linear relationship between the total amount of water remaining in a pool filtration tank, $W$, in gallons, and the number of minutes, $m$, since a drainage pump started running.Based on the graph, what is the initial volume of water inside the tank before the pump began running?

In the standard xy-plane, the graph of the linear function f(x)=mx+b is a line that has a positive slope and a negative y-intercept. Which quadrant of the coordinate plane contains NO points that satisfy this function?

The graph of the linear function $f(x) = -\frac{2}{3}x + k$ is plotted in the standard $xy$-plane, where $k$ is a positive constant. If the line is shifted vertically downward by 5 units, which of the following statements describes the transformation seen on the new graph?

In the standard $xy$-plane, a linear equation is graphed. The line passes through the point $(2, -3)$ and has a slope of 0. Which of the following is the correct equation represented by this graph?

The complete graph of the linear function g is shown in the standard xy-plane. The line passes through the coordinates (−4,1) and (2,4). What is the y-intercept of the graph of g?

A business analyst graphs a line that models a store's daily operating net cost profile. The horizontal axis represents the number of items sold, and the vertical axis tracks total expenses in dollars. If the graph has a vertical intercept at $(0, 1200)$ and a horizontal intercept at $(80, 0)$, what does the horizontal intercept represent in this context?

In the standard $xy$-plane, the graphs of the linear functions $f(x) = ax + 4$ and $g(x) = bx - 6$ are perpendicular lines that intersect at a point located on the horizontal $x$-axis. If $a > 0$, what is the value of the product $ab$?

Line $\ell$ is graphed in the standard $xy$-plane. It has a negative slope and a positive $y$-intercept. If line $k$ is created by reflecting line $\ell$ completely across the horizontal $x$-axis, which of the following statements must be true about the graph of line $k$?

In the standard $xy$-plane, a linear function graph contains the coordinate points $(k, 3)$ and $(2k, 11)$, where $k$ is a non-zero constant. If the horizontal intercept ($x$-intercept) of this same line is located at $(-2, 0)$, what is the value of the constant $k$?
