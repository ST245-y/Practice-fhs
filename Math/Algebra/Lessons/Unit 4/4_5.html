Lesson 5: No Solution and Infinite Solutions in Systems

$$\begin{cases} 3x - 4y = 12 \\ 3x - 4y = k \end{cases}$$In the given system of equations, $k$ is a constant. If the system has no solution, which of the following could NOT be the value of $k$?

$$\begin{cases} y = 5x + 3 \\ y = ax + 7 \end{cases}$$In the given system of equations, $a$ is a constant. If the system has no solution, what is the value of $a$?

$$\begin{cases} 2x - 5y = 8 \\ 6x - 15y = c \end{cases}$$In the given system of equations, $c$ is a constant. If the system has infinitely many solutions, what is the value of $c$?

$$\begin{cases} ax + 12y = 9 \\ 2x + 3y = 4 \end{cases}$$In the given system of equations, $a$ is a constant. If the system has no solution, what is the value of $a$?

$$\begin{cases} \frac{1}{2}x - \frac{2}{3}y = 5 \\ ax - 4y = 30 \end{cases}$$In the given system of equations, $a$ is a constant. If the system has infinitely many solutions, what is the value of $a$?

$$\begin{cases} y - 4 = k(x + 1) \\ 2x - 3y = 7 \end{cases}$$In the given system of equations, $k$ is a constant. If the system has no solution, what is the value of $k$?

$$\begin{cases} px + qy = 10 \\ 3x + 4y = 5 \end{cases}$$In the system of equations above, $p$ and $q$ are constants. If the system has infinitely many solutions, what is the value of the sum $p + q$?

$$\begin{cases} (k-1)x + 2y = 4 \\ 3x + ky = 7 \end{cases}$$In the given system of equations, $k$ is a constant. If the system has no solution, which of the following could be the value of $k$?

$$\begin{cases} 4x - b = 2y \\ ax + 3y = 12 \end{cases}$$In the system of equations above, $a$ and $b$ are constants. If the system has infinitely many solutions, what is the value of the product $ab$?

$$\begin{cases} c = \frac{2}{5}x + \frac{1}{3}y \\ 4 = dx - 2y \end{cases}$$In the given system of equations, $c$ and $d$ are constants. If the system has infinitely many solutions, what is the value of the constant product $cd$?
